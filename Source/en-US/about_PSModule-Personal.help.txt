SHORT DESCRIPTION
    A personal module.

LONG DESCRIPTION
    This project is to build a personal module using GitHub (https://github.com/) 
    and use the suggested ModuleBuilder (https://github.com/PoshCode/ModuleBuilder) way of working.

    - Build a simple set of handy tools.
    - Some scripts are based on tips & trick by other scripters that will be mentioned in the script.
    - For more details see:
        - https://github.com/gitoldi/PSModule-Personal
        - And use 'Get-Help <command>' for individual scripts.

KEYWORDS
